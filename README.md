## Features

* Add new Projects.
* Add or reslove issues.
* color coded labels.
* Persistant redux state.
* Search based Projects filter.
* Search based issue filter.

## How to install and run?

1. Clone the repositories.
2. Install all dependencies by `npm install`
3. `npm start`
